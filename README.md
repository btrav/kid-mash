Toddlers want to use the keyboard. This gives them somewhere to do it — full-screen, colorful, and impossible to break. Every letter has its own emoji (R = 🤖, G = 🦒, A = 🐊 ...), every press triggers an animation, and spacebar launches fireworks.

## Toddler mode 🧸

Hit the 🧸 button for a calmer experience: one consistent emoji per letter, gentle fade-in animations, one firework at a time, no streak banners. Good for younger kids or quieter moods.

## What's included

- 26 letters × 5 emojis each — a different character for every letter
- 8 animations: bounce, spin, float, wobble, pop, punch, drop, gentle fade
- Spacebar: fireworks. Hold for rapid-fire. Every 20–40 presses triggers a mega firework
- Sound mode — Web Audio API synth tones, off by default
- Combo banner on 35-press streaks
- Mouse drag trail, click counter, key counter
- Light/dark theme toggle
- Toddler mode: calmer everything

## Stack

- [Vanilla JS](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — no framework, no build step
- [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API) — synthesized sounds, no audio files needed
- [Google Fonts – Fredoka One](https://fonts.google.com/specimen/Fredoka+One)
